# Dotfiles  

My personal configuration files for macOS/Linux. Includes setup scripts and configs for shell, editors, and more.  

## Installation  

Clone and symlink:  
```bash
git clone https://github.com/dragospascariu/dotfiles.git
cd ~/.dotfiles
```

## Dependencies and extras
- `homebrew`
- `fzf`  
- `fd`  
- `ripgrep`  
- `zoxide`  
- `eza`  
- `tldr`  
- `mas`
